### 算法时间复杂度

1.算法的时间复杂度是程序运行的从开始到结束所需要的时间。

```C
float sum(float list[],const int n)
{
  float tempsum = 0.0;
  for(int i = 0; i < n ; i++)
  tempsum += list[i]
  return tempsum
}
```
